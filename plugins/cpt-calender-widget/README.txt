== Installation ==

1. Upload the entire `CPT Calender Widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. drag and drop CPT Calender widget in sidebar area
4. Select your post from dropdown
5. Call that sidebar in your page/post
6. enjoy!
